The Mary Russell Mitford Archive:

Background

The Digital Mitford Archive is an ongoing project started by Dr Beshero-Bondar, of Penn State Behrend, almost ten years ago about the life of Mary Mitford. Mary Mitford has lived in a long and enriched life in the 1800's, with plenty of documentation. Therefore, the project was formed with the Mary Russell Mitford Society in the 18th- and 19th-century British Women Writers’ Conference in Albuquerque. So far there have been many side projects completed and linked on the main page.

The Project

We are a group of four students in the class of Text Encoding for DIGIT Majors and have come together to work collectively on the Digital Mitford Project. In this site, we will be pulling in our own smaller projects to show off each other’s work using information in Mary Mitford’s journal entries. Tyler Cameron created the Plants & Animals page, Jacob Huellen helped with research and photo references, Savannah Ricks created the Home, Hayward, and About pages, and Tyler Akam created the 1819 and 1820 Entries pages. This project is based off learning many encoding languages in the class DIGIT 110, such as: XML, RelaxNG, TEI, ODD, XSLT, and some HTML/CSS. We learned a lot in this class, and we hope you've learned something too from our project!

For a view of the code we used and process we've taken, go to our GitHub repo!
